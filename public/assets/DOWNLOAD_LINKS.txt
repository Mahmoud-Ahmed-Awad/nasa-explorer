------------------------------------------------------
Download these TWO files from Three.js GitHub:

   URL: https://raw.githubusercontent.com/mrdoob/three.js/dev/examples/textures/lensflare/lensflare0.png

2. lensflare3.png
   URL: https://raw.githubusercontent.com/mrdoob/three.js/dev/examples/textures/lensflare/lensflare3.png

Quick Setup Steps
-----------------
1. Download lensflare0.png and lensflare3.png (CRITICAL!)
2. Download or create default.jpg for each category
3. Gradually add specific images as needed
4. Run: node scripts/download-nasa-images.js (if using Node script)


Minimum Viable Setup (3 files)
-------------------------------
To get the app running with minimal images:

1. public/assets/lensflare/lensflare0.png  ← MUST HAVE
2. public/assets/lensflare/lensflare3.png  ← MUST HAVE
3. Any other images can use defaults or fallback gracefully
